# Changelog

## Version 1.0.0

- First maintained public GitHub release.
- Carries forward the 0.0.4 content base with Codex maintenance fixes, Nexerelin setup, Version Checker/TriOS metadata, weapon hardening, and runtime optimization work.

## Version 0.0.4

### Codex maintenance build

- Added Version Checker/TriOS metadata.
- Added Nexerelin faction-generation configs for the corporate factions.
- Polished faction presentation, descriptions, and lore consistency.
- Fixed custom weapon projectile/effect wiring issues.
- Hardened missile AI and EMP on-hit scripts against null target/source edge cases.
- Optimized Delicacy fleet support synchronization so it no longer scans every fleet every frame.
- Stopped registering legacy no-op income scripts on new loads.
- Fixed fighter listener cleanup for Luddic Air Control.
- Converted Zenith luxury support behavior away from direct fleetwide income and toward ship-local maintenance benefits.
- Improved colony item compatibility with installed industry libraries.

- Added Energy Swarm Core tactical support.
- Added Predator, Predator EB, Base Stage, Dissection, Short Gun, and Qingguang fighter content.
- Added Dire Wolf high-speed destroyer.

- Added Ionized Rechargeable Armor.

- Reduced corporate tariffs and revised market naming.
